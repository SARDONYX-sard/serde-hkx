Attribute Macro impl_flags_methods

#[impl_flags_methods]
Expand description

Automatically implement [core::str::FromStr], [core::fmt::Display] in string notation according to Havok’s hkFlags<Enum, SizeType> XML notation.

In addition, [core::default::Default] is implemented.
