Histogram diff algorithm.

This implementation follows the idea behind Git’s histogram diff: find a common region anchored on low-frequency elements and recurse around it. Compared to plain Myers this tends to prefer structurally meaningful anchors over very frequent lines.

  • time: input-dependent (typically close to Hunt/Patience-style behavior); worst-case falls back to Myers (O((N+M)D))
  • space: O(N + M) plus match-list/index overhead
